# Portfolio — Buzau Cosmin-Ionut

Personal portfolio site: hero, about, skills, projects and contact sections, built as a single-page Next.js app.

## Tech Stack

Next.js 16 (App Router) · React 19 · TypeScript · Tailwind CSS 4 · Framer Motion

## Structure

- `src/data/portfolio.ts` — all content (profile, skills, projects, achievements) in one place
- `src/components/` — Navbar, Hero, About, Skills, Projects, Contact, Footer
- `public/profile.png` — profile photo
- `public/CV_Buzau_Cosmin_Ionut.pdf` — downloadable CV, linked from the navbar and hero

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
